# Rust-Caffeine ☕

Windows 시스템의 절전 모드와 화면 꺼짐을 방지해 주는 초경량 시스템 트레이 애플리케이션입니다.
기존 파이썬(Python) 기반의 Caffeine 프로그램을 **Rust로 완전히 재작성**하여 메모리 사용량을 수 십 분의 일 수준(약 1~3MB)으로 극적으로 줄인 네이티브 버전입니다.

## ✨ 주요 기능 및 특징

- **초경량 시스템 포맷:** Rust 기반 네이티브 빌드로 제작되어 백그라운드에서 동작 시 램(RAM) 자원을 거의 소모하지 않습니다.
- **윈도우 절전 완벽 방어:** Windows API (`SetThreadExecutionState`)를 직접 호출하여 마우스를 흔들거나 동영상을 켜놓지 않아도 화면이 꺼지거나 PC가 잠자고 있는 것을 방지합니다.
- **단일 실행 파일:** 실행에 필요한 아이콘 에셋 파일들이 빌드 시퀀스 중 `.exe` 내부에 직접 포함(Embed)됩니다. 배포 및 실행 시 이 1개의 `.exe` 파일만 있으면 됩니다.
- **콘솔(터미널) 숨김:** 불필요한 콘솔 창이 뜨지 않으며, 더블클릭 시 즉시 시스템 트레이(우측 하단)에서만 조용히 작동합니다.

## ⚙️ 아키텍처 및 라이브러리 (Dependencies)

- **언어:** Rust
- **UI 및 트레이 제어:** `tray-icon`, `tao`, `muda` (Tauri 팀 표준 트레이 모듈 사용)
- **Windows 시스템 콜:** `windows-rs` 크레이트를 이용한 `Win32::System::Power` 통신
- **이미지 파싱:** `image` 크레이트를 이용한 실시간 `.ico` 파일의 RGBA 바이트 분석 및 메모리 로딩

## 📖 사용 방법

1. `rust-caffeine.exe` 파일을 더블클릭하여 실행합니다. (권한 요청이 불필요합니다.)
2. Windows 우측 하단의 **시스템 트레이(상태 표시줄)** 영역에 주황색 커피잔 모양의 아이콘이 나타납니다.
   - 🟠 **주황색 아이콘 및 김 모락모락:** 현재 잠금 방지 기능이 **활성화**된 상태입니다.
   - ⚪ **회색 아이콘 및 빈 컵:** 기능이 **비활성화**되어 일반적인 윈도우 절전 정책을 따르는 상태입니다.
3. 아이콘을 **우클릭**하여 나오는 메뉴에서 다음을 선택할 수 있습니다.
   - `Caffeine 켜기/끄기`: 절전 방지 활성/비활성 상태를 토글합니다.
   - `종료`: 프로그램을 메모리에서 완전히 해제하고 시스템 절전 모드를 원래대로 복구합니다.

## 🛠️ 자체 빌드 방법 (How to Build)

만약 코드를 직접 수정하거나 소스 코드로부터 새로 컴파일해야 한다면 아래 과정을 따라주세요.

### 1. 요구 사항 (Prerequisites)
- [Rust & Cargo](https://rustup.rs/) 설치 (최신 버전 권장)
- Windows 환경 (Win32 API를 사용하므로 Mac/Linux 빌드는 지원되지 않습니다.)

### 2. 빌드 스텝

명령 프롬프트나 터미널을 열고 본 프로젝트 폴더(`rust-caffeine`)로 이동한 뒤, 아래 명령어를 실행합니다.

```sh
# 릴리즈 모드(가장 최적화되고 가벼운 모드)로 컴파일합니다.
cargo build --release
```

### 3. 결과물 실행
컴파일이 완료되면 폴더 내의 `target\release` 경로에 실행 파일이 생성됩니다.
```sh
# 실행 파일 경로
.\target\release\rust-caffeine.exe
```
